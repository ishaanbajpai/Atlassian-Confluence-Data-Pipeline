Editor usecase for Developer community- JS

In Javascript today we write lot of functions like F.onLoad(), F.<Field>.onChange(),F.<Multirow>.onAddRow(),etc.,

On those functions, developer used to write UI related manipulations such that

  1. F.onLoad()

    1. Make a field with some default value if form loads first time(stage= CREATE_EDIT).

    2. call some report to render in the screen

    3. do some calculation, loading reusable component or complex operation by calling some custom function

    4. Hide or show few sections.

  2. F.OnChange()

    1. make another dependent field to be blank

    2. doing some calculation or logic by calling custom function

  3. F.<Field>.onAddRow()

    1. Show and hide fields

    2. make fields readonly or editable

    3. Make some fields defaulted with some value while adding only

    4. doing some calculation or logic by calling custom function

These are few examples..There are other functions also like that we does the screen manipulation. All that needs to be addressed.
